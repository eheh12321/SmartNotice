# 스마트 마을알림 시스템
스마트 마을 알림 시스템은 마을들에 대한 종합적인 관제 서비스를 제공하면서 각 마을에 문자/음성 방송 송신, 사용자 긴급 호출 수신 등 시골 마을 또는 독거 노인 관리 서비스를 제공하는 시스템입니다.
